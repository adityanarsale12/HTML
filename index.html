Bold, Italic & Underline Tags
Used to highlight text in your page.
< b > Bold < /b >
< i > Italic < /i >
< u > Underline < /u >

Big & Small Tags
Used to display big & small text on your page.
< big > Big < /big >
< small > Small < /small >

Hr Tags
Used to display a horizontal ruler, used to separate content.
< hr > (Space comes with line)

Subscript & Superscript Tags
< sub > Subscript < /sub >
H₂O (< p > H < sub > 2 < /sub > O < /p> )

< sup > Superscript < /sup >
Aⁿ + B (< p > A < sup > n < /sup > +B < /p> )

Pre Tag
Used to display text as it is(without ignoring spaces & next line)
< pre > This 
            is a sample
            text 
< /pre >
जब खुदका space चाहिये तो < p > नही < /pre > use करे!
